TIC TAC TOE

A GUI enabled, multi-player, network facilitated Tic-Tac-Toe Game in Java using Object Oriented Programming and multi-threading.

Instructions to start the game:

-Open command line terminal and navigate to this folder's location.
-Type in the command 'javac *.java' to compile all the .java files.
-Type in the command 'java TicTacToeServer' to launch the game Server.
-Open two more command line terminal tabs.
-On each of them, first navigate to this folder's location and then type in the command 'java Client' to initiate the two Clients.
